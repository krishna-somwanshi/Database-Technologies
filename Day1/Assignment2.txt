4) Why isn't it possible to see the first five rows of a table?
Ans:- Yes, it is true that we can not see first five rows of a table. The reason is that when we try to Access the first five rows in relational Databases it will give us the first five memory location.
	Since it gives us data from first five memory address so we can not determine that output data is from first five rows only. It will gives us output of first five memory locations.
